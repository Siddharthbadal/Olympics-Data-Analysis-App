## Olympics Data Analysis Web App

Goal of the project is to analyze olypics data of over 100 years on defferen levls, and use charts and graph to support data points.

Data Analysis Process
  - Data Preprocessing
    - Data cleaning, organizations 
  - Data Analysis
    - Analysis using pandas. Data segmentations and categorizing data points, data processing by creating mutiple dataframes. Creting function for each data point. 
  - Data Visualizations
    - Used matplotlib and plotly for data visualization. Used line chart, bar chart, pie charts.
  - Web App Development 
    - Prepared a streamlit app, pushed to github and hosted it on streamlit cloud. 

Data Analysis is perfomed on various levels
  - Overall Analysis
  - Country Analysis
  - Sports Analysis
  - Athlete Analysis
  - Yearly analysis

Modules Used
  - Pandas
  - streamlit
  - plotly
  - seaborn
  - matplotlib

DataSet: Dataset Link: https://www.kaggle.com/heesoo37/120-years-of-olympic-history-athletes-and-results
